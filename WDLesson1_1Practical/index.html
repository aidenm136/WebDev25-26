Web Technologies

HTML

HTML stands for hypertext markup langauge. HTML is the standard markup language used to create and structure web pages and their content. Used to understand programs

CSS

CSS stands for Cascading Style Sheets. A style sheet language used for describing the presentation of a document. It is a fundamental technology for web development

JavaScript

JavaeScript is an object-oriented computer programming language commonly used to create interactive effects within web browsers. Primarily known for enabling interactive and dynamic content on websites. It is a core technology of the World Wide Web

HTTP

Stands for Hypertext Transfer Protocol. It's the foundational application-layer protocol of the World Wide Web. Defines how web browsers and web servers communicate by sending and receiving hypermedia documents like HTML
